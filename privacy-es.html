Política de privacidad

DroneKoll — última actualización: 2026-04-07

Resumen

DroneKoll no recopila ninguna información personal sobre ti. No vendemos, compartimos ni almacenamos ningún dato en servidores externos.

Datos de ubicación

La aplicación utiliza tu posición GPS para mostrar información meteorológica y tráfico aéreo relevante en las proximidades. Tu ubicación nunca sale de tu dispositivo: no se envía a ningún servidor ni se almacena de forma permanente.

La aplicación realiza solicitudes API anónimas a servicios abiertos (Open-Meteo, adsb.fi, Nominatim/OpenStreetMap) utilizando tu posición aproximada. Estos servicios pueden registrar direcciones IP según sus propios términos.

Registro de vuelos

Los vuelos registrados (hora, ubicación, dron) se almacenan solo localmente en tu dispositivo mediante el almacenamiento estándar de iOS. Ningún dato se sincroniza con la nube ni se comparte con terceros.

Certificado de dron

Las imágenes que subes se almacenan solo localmente en tu dispositivo.

Ajustes y preferencias

Los ajustes de la aplicación (por ejemplo, país seleccionado, modelo de dron) se almacenan localmente en la configuración de la aplicación en tu dispositivo y no se transmiten.

Servicios de terceros

La aplicación se comunica con los siguientes servicios externos. Estos servicios procesan las solicitudes (que pueden incluir direcciones IP) de acuerdo con sus propias políticas de privacidad:

  • Open-Meteo — pronósticos meteorológicos
  • adsb.fi — tráfico aéreo en tiempo real (ADS-B)
  • OpenStreetMap / Nominatim — mosaicos de mapa y geocodificación
  • RainViewer — radar de precipitaciones
  • FAA Aviation Weather Center — información NOTAM

No se transmiten datos personales a estos servicios. Las solicitudes solo contienen coordenadas geográficas para obtener datos basados en la ubicación.

Sin seguimiento ni analítica

La aplicación no utiliza ningún servicio de analítica, seguimiento o publicidad. No se crean perfiles de usuario.
